What is Sin?

Brought to you by the LinkedIn content search team. It is an application that hosts content search and storage services. Sin is built on top of the Sensei technology. Sin hides the complexities of Sensei while exposing its power.

Why Sin?

  • No need to do machine provision
  • Quick setup, can your content system running in just minutes!
  • Focus on your application instead of engineering issues in building a content system

Features:

  • Simple API - Java & Python
  • Simple web-based administration
  • Fast-retrieval via document id
  • Full-text search
  • Faceted search
  • Real-time update
  • Sharding - Distributed index
  • Distributed commit log
  • Partial updates
  • Hadoop integration
