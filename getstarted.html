Quick Start:

  1. Create a new store from the Dashboard, with a name, say myidx
  2. Start the store
  3. Submit a doc to it by: 
    curl -X POST 'http://sinHost:sinPort/store/add-docs/myidx' -d 'docs=[{"id":1,"contents":"sin is awesome"}]' -i
  4. Refresh the Dashboard, you should see store size increased
  5. Now you can search using the Search Endpoint, e.g. 
    curl -X GET 'http://searchHost:searchPort/sensei?q=awesome&start=0&rows=10&fetchstored=true' -i
