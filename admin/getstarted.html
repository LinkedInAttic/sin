Quick Start:

  • Create a new store from the Dashboard, with a name, say myidx
  • Start the store
  • Submit a doc to it by: 
    curl -X POST 'http://sinHost:sinPort/store/add-docs/myidx' -d 'docs=[{"id":1,"contents":"sin is awesome"}]' -i
  • Refresh the Dashboard, you should see store size increased
  • Now you can search using the Search Endpoint, e.g. 
    
curl -X GET 'http://searchHost:searchPort/sensei?q=awesome&start=0&rows=10&fetchstored=true' -i
